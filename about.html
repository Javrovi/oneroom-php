About OneRoom

{% endblock %} {% block content %}

OneRoom is a simple grade tracking web application that I wrote for my programming portfolio in summer 2011. This version of OneRoom is written using the Django web application framework. There are two other versions of OneRoom in my portfolio: one written with the Rails framework, and another one written with the LAMP stack (PHP and MySQL). I coded OneRoom as a simple (but useful) web app that I can try out different frameworks with for comparison and contrast. In the future I may write more versions of OneRoom (e.g., Groovy and Grails?).

What does OneRoom do?

OneRoom is an application that allows teachers and students to track grades. Teachers can input grades for assignments and students can view their grades. For more details, go to the Help page. OneRoom uses the MVC (model-view-controller) design pattern. There is a little bit of AJAX in OneRoom - the search function is a live search.

What languages/frameworks/standards are used in OneRoom?

In this version of OneRoom, I used the Django web application framework (version 1.3), which is written in the Python language. For the AJAX live search feature, I used the jQuery Javascript library. In addition, I also used CSS for formatting the (XHTML) web pages.

What resources did you use in coding and designing OneRoom?

The online documentation for Django and jQuery are both excellent. In addition,

  • Django 1.0 Web Site Development from Packt Publishing, also available on Safari Books
  • CSS: the Missing Manual from O'Reilly Publishing, also available on Safari Books.
  • The color scheme for OneRoom's web pages is based on the Giant Goldfish color palette. It is the most "loved" palette on ColourLovers.
